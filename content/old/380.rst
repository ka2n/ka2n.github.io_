追加した要素にイベントハンドラ
##############################
:date: 2010-10-21 03:52

:slug: 380

脳を溶かすjqueryネタ。

.. code-block:: javascript
   
    $('#hige > a').click(function(){
        alert('hage');
    });

なんて書いてあっても、

.. code-block:: javascript

    ('#hige').append('<a href=".">追加</a>');

で追加したやつにはイベントが設定されていないので、反応しない。
そういう場合は、

.. code-block:: javascript

    $('#hige > a').live('click', function(){
    alert('hage');
    });

と書く。ふむふむ。
